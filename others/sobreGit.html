Git Primer Contacto

Git es un repositorio versionador, muy utilizado en el mundo del desarrollo web y la programación

En primer lugar se debe descargar el programa git desde su página oficial en https://git-scm.com/

Seguimos el procedimiento de descarga e instalación habitual y verificamos si todo se instaló correcctamente abriendo un terminal y ejecutando el comando git version

si todo fué correcto el comando nos informa algo como git version 2.25.1

Para iniciar Git dentro de un proyecto debemos abrir un terminal de consola (Terminal - Nuevo Terminal), en la barra de comandos aparecerá algo como administrador@Administrador$: en esa linea escribimos git init y damos enter
